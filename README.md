# 📈 Global Markets Analytics Dashboard

A Python-based financial analytics dashboard that uses real stock market data to visualize price trends, compare companies, and analyze market performance.

## 🚀 Project Overview

## 📊 Dashboard Preview

![Stock Dashboard](images/dashboard.png)
This project collects historical stock market data and creates visual insights using Python. It allows users to analyze different companies and understand price movements through data visualization.

The goal of this project is to build a simple version of a financial analyst tool used to explore market trends.

## ✨ Features

- 📊 Download real stock market data
- 📈 Visualize historical stock prices
- 🔍 Compare multiple companies
- 💹 Analyze closing price trends
- 🖥️ Create financial charts

## 🛠️ Technologies Used

- Python
- yFinance (Yahoo Finance API)
- Pandas
- Matplotlib
- Git & GitHub
